指令列表

一般/未分類​

指令功能
?ping查看延遲
?icon　@成員查看該成員的頭像
?guildinfo查看伺服器的資訊
?userinfo @成員查看該成員的資訊
?nitro這是一個隨機產生器(連結基本上是沒用的哦)

搜尋​

指令功能
?google <你要搜尋的東西>沒用的指令
?covid查看全球病毒的資訊
?bs <#TAG>查看該玩家在荒野亂鬥的資訊
?hypixel <玩家名稱>查看該玩家在hypixel伺服器的資訊
?steam <遊戲商店連結>查看該遊戲的資訊
?yt <YouTube連結/關鍵字>搜尋Youtube影片

音樂​

指令功能
?play <YouTube連結/關鍵字>播放歌曲
?nowplay查看目前播放的資訊
?pause暫停音樂
?resume回復播放
?stop停止音樂(無法回復播放)
?skip跳過音樂
?volume <1-200>設定音量(預設:100)
?dc中斷語音頻道
?join加入語音頻道
?loop單曲重複播放
?loopall播放清單重複播放
?nowplay查看目前播放的資訊

管理員​

指令功能
?clean <刪除多少信息(1-20)>刪除該頻道的信息
?kick @成員剔除成員
?ban @成員封鎖成員

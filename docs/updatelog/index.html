更新日志

最新版本: v0.3.3b

版本: 0.3.3b / 2022-2-24​

新增​

  • ?hypixel <玩家名稱>
    • 查看該玩家在hypixel伺服器的資訊
  • ?bs <#TAG>
    • 查看該玩家在荒野亂鬥的資訊
  • ?steam <遊戲商店連結>
    • 查看該遊戲的資訊
  • ?timeout @成員 <時間> [原因(可選)]
    • 禁言成員
  • ?removetimeout @成員 [原因(可選)]
    • 解除禁言成員
  • 部分指令設有斜綫(/)指令的版本
    • /sguildinfo
    • /shelp
    • /snowplay
    • /sping
    • /sremovetimeout
    • /stimeout
    • /svolume

修改​

  • ?play
    • 如果你後面輸入的是關鍵字，你會看到搜尋結果由三個按鈕變成下拉式選單
    • 多了一項錯誤偵測: 無法搜尋結果

版本: 0.3.2b / 2021-12-20​

新增​

  • ?play
    • 如果?play後面不是youtube連結會彈出搜尋結果有前三個搜尋結果可以選擇
  • YouTube播放指令支援更多類型的連結
    • https://music.youtube.com/watch?v=...
    • http://music.youtube.com/watch?v=...
    • https://youtu.be/...

版本: 0.3.1b / 2021-08-12​

新增​

  • ?loop - 重複播放音樂功能
  • ?cplay <連結> - 自定義播放 (BETA)
    • 連結必須是音樂檔案的連結 (例如https://www.example.com/song.mp3 / 暫時只支援MP3檔案)
  • YouTube播放指令支援更多類型的連結
    • https://www.youtube.com/watch?v=......
    • http://www.youtube.com/watch?v=......
    • http://m.youtube.com/watch?v=......
    • https://m.youtube.com/watch?v=......
    • http://youtube.com/watch?v=......
    • https://youtube.com/watch?v=......
    • youtube.com/watch?v=......
    • www.youtube.com/watch?v=......
    • m.youtube.com/watch?v=......

修改​

  • 取消了?playlist/?playload/?lyrics/?soundcloud/?myinfo指令

已知BUG​

  • ?loop - soundcloud無法一直重複播放 (YouTube可以)

版本: 0.3.1b (Patch 1) / 2021-08-14​

  • ?play - Soundcloud連結如果是播放清單會告訴錯誤信息並列表清單内的音樂與網址，讓大家可以打?p <網址> 支援播放清單的功能應該會在未來更新
  • ?nowplay - 會顯示下一首的歌曲

版本: 0.3.1b (Patch 2) / 2021-08-29​

  • ?skip 的BUG已修復(skip不是到下一首歌曲)
    • 全部音樂的音量都很貼心的減低一倍(原本的太大聲了)

版本: 0.3.1b (Patch 3) / 2021-12-04​

  • ?cplay 新增支援播放mp4, aac, wav的網址
